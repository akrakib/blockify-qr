# Blockify QR Tool

[🔗 Live Demo](https://akrakib.github.io/blockify-qr/)

Blockify QR Tool is a **modern web-based QR code generator** and **scanner** with a **futuristic UI**, particle animation background, and **local history support**. It is designed with **HTML**, **CSS**, and **JavaScript** for a seamless experience.

### Features:
- **QR Code Generation**: Instantly generate QR codes with custom URLs or text.
- **QR Code Scanning**: Use your webcam or upload an image to scan QR codes.
- **Particle Animation Background**: A visually appealing particle animation for an interactive experience.
- **Local History Support**: Keep track of previously generated QR codes for quick access.
- **Fully Responsive**: Works perfectly across all devices (desktop, tablet, mobile).

### Tech Stack:
- **Frontend**: HTML, CSS, JavaScript
- **Libraries**: 
  - [QRCode.js](https://github.com/davidshimjs/qrcodejs)
  - [Particle.js](https://github.com/VincentGarreau/particles.js)
